# Saarathi – Frontend Web Application

This project is a modern frontend web application built using **TypeScript, HTML, CSS, and Tailwind CSS**, bundled with Vite for fast development and performance.

## 🚀 Features
- Responsive and clean UI design
- Component-based frontend structure
- Fast development with Vite
- Styled using Tailwind CSS
- Organized code using TypeScript

## 🛠️ Tech Stack
- TypeScript
- JavaScript
- HTML
- CSS
- Tailwind CSS
- Vite
